# SQL Constraints
## SQL Create Constraints
create table에 붙일 수 있는 constraints

```sql
CREATE TABLE table_name (
    column1 datatype constraint,
    column2 datatype constraint, 
    column3 datatype constraint
);
```

## SQL Constraints
- NOT NULL
- UNIQUE
- PRIMARY KEY
- FOREIGN KEY
- CHECK
- DEFAULT
- CREATE INDEX
